# 🔐 Penetration Testing for Banking Login System

## 📌 Project Overview
This project demonstrates *penetration testing on a banking login system* inspired by real-world scenarios similar to HSBC.  
The goal is to identify security vulnerabilities and implement fixes using secure coding practices.

---

## 🎯 Problem Statement
Banking applications handle sensitive user data. If security is weak, attackers can exploit vulnerabilities such as SQL Injection and gain unauthorized access.

---

## 💡 Solution
A login system is developed using PHP and MySQL.  
Penetration testing is performed to:
- Identify vulnerabilities
- Exploit them
- Apply fixes to secure the system

---

## 🛠️ Technologies Used
- PHP  
- MySQL  
- HTML & CSS  
- XAMPP Server  
- Zenmap (Network Scanning Tool)

---

## ⚙️ Features
- User login system  
- SQL Injection vulnerability demonstration  
- Secure login using prepared statements  
- Basic input validation  
- Network scanning using Zenmap  

---

## 🚨 Vulnerabilities Identified
- SQL Injection  
- Weak Password  
- Open Ports  

---

## ✅ Fixes Implemented
- Used prepared statements to prevent SQL Injection  
- Updated password to strong format  
- Added input validation  
- Suggested closing unused ports
